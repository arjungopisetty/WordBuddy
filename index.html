WordBuddy

Thesaurus Extension with cross-application compatibility that integrates with Leap Motion. It can be useful for many applications that don't have this functionality, for example, Pages or a simple text editor. With Leap Motion, a user can click a word, make a finger gesture, and dynamically get the list of synonyms for that word to pop up on the screen. Then, the user can substitute these words into the document seamlessly without pressing any buttons.
